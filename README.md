## How to Run
1.  Clone this repository to your local machine.
2.  Navigate into the project directory.
3.  Create and activate a Python virtual environment:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
4.  Install the necessary dependencies:
    ```bash
    pip install pandas
    ```
5.  Run the script:
    ```bash
    python3 cleaner.py
    ```
